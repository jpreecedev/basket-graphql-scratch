# GraphQL with .NET back end PoC

A test project for understanding how to use GraphQL with a .NET back end

- Webpack 4
- Babel 7
- Jest
- Enzyme
- ESLint & Prettier
- Generator polyfills
- GraphQL with Apollo

## Main commands

```shell
# Run the dev server on port 8080
yarn start

# Generate a production build
yarn build

# Run ESLint
yarn lint

# Run unit tests
yarn test
```
